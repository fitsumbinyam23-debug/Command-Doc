# Learning Experience UI Standard

The learning UI is beginner-first, bright, responsive, and explicit about planned content.

- Home primary action: Start Level 0
- Course map: current phase first, future phases collapsed by default, locked levels show prerequisite reasons.
- Level overview: Level 0 is authored; later levels are labeled planned.
- Lesson stepper: Mission, Learn, See, Key words, Predict, Try, Explain, Confidence, Continue
- Technician Tools: Diagnose, Command Lookup, Focused Terminal, Guided CLI, Switch Workbench, Visual Playground, Practice Library, Knowledge Base, Saved Reports

Controls preserve progress, avoid fake mastery, and keep Instructor Mode out of beginner navigation.
