# Beginner Learning Experience

The beginner experience gives a first-time learner three clear journeys: Learn From Zero, Practise and Specialize, and Technician Tools.

## Navigation

Beginner navigation contains: Home, Course, Practice, Progress, Tools. Instructor Mode is intentionally absent from beginner navigation.

## Level 0

Level 0 uses the stepper sequence: Mission, Learn, See, Key words, Predict, Try, Explain, Confidence, Continue. It records local orientation progress and confidence only. It does not award command mastery.

## Progress

The first-run choice and Level 0 resume state use dedicated browser storage, preserving existing lab progress, vendor progress, reports, and technician tool history.
