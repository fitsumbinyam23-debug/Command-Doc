## Current Product Shape
- Home is the resume/orientation surface, but it combines completion, mastery, review, and route progress without a single evidence model.
- Diagnose is the strongest evidence-first flow and should remain fast and technician-oriented.
- Learn covers the whole command catalog through generated modules, but premium lesson behavior exists only for a few commands and is not consistently runtime-backed.
- Switch Lab contains the runtime-rich experience and should become the proving ground for practical execution, verification, Save, and rollback.
- Library is useful but currently mixes reference, practice, reports, and instructor tools; v0.3.0 should make it support the journey without becoming the journey.

## Disposition Summary
- Keep: Diagnose, Command Lookup, Knowledge Base, Focused Terminal.
- Improve: Home, Switch Lab, Practice Library, Saved Reports, Guided CLI, Switch Workbench, Visual Playground, vendor selection, progress indicators, navigation, empty/error states, mobile, accessibility.
- Replace after migration: Learn lesson execution, foundation/configuration lesson completion, mastery indicators, review indicators.
- Hide or gate: Instructor Mode in student-facing flows.
